你这个数据集最推荐这样跑：

cd /root/autodl-tmp/paraformer-arc

python convert_dataset_to_paraformer.py \
  --project-root /root/autodl-tmp/paraformer-arc \
  --dataset-root /root/autodl-tmp/paraformer-arc/chinese_ATC_formatted \
  --wav-subdir WAVdata \
  --txt-subdir TXTdata \
  --output-dir FunASR/data/list \
  --path-mode abs \
  --strip-wav-suffix _16k \
  --also-generate-jsonl

跑完后会直接得到：

FunASR/data/list/train_wav.scp
FunASR/data/list/train_text.txt
FunASR/data/list/val_wav.scp
FunASR/data/list/val_text.txt
FunASR/data/list/train.jsonl
FunASR/data/list/val.jsonl
